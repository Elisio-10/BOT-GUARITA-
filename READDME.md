# 🤖 Bot de Gestão Startup (WhatsApp)

Bot avançado para gerenciamento de grupos com sistema de pareamento via código.

## 📋 Comandos
- `.abrir` : Abre o grupo para todos.
- `.fechar` : Fecha o grupo para administradores.
- `.kick` / `.ban` : Remove o usuário marcado.
- `.link` : Envia o link do grupo.
- `.informações` : Mostra dados do grupo.

## 🚀 Como instalar (Mobile)
1. Crie um projeto Node.js no **Replit**.
2. Cole estes arquivos.
3. No console, digite `npm install`.
4. Digite `npm start`, coloque seu número e use o código de 8 dígitos no seu WhatsApp.
